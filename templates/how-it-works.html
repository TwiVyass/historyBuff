How HauteBot Works

Architecture Overview

User input is sent through a Flask backend, which interacts with OpenAI's GPT and image generation models (like DALL·E). The response is parsed and shown as conversational messages — or concept sketches if requested.

Stack:

  • Frontend: HTML, CSS (glam-themed)
  • Backend: Flask (Python)
  • APIs: OpenAI Chat, Image Generation

Sample Flow

“Draw a modern gown inspired by 1920s Art Deco.”
→ Generates a sketch + details about geometric cuts, metallic palettes, and dropped waistlines.
